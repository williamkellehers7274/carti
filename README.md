# Discord бот заявок (Embed v2) — Python

Бот отправляет панель с выбором типа заявки (**ACADEMY** / **YOUNG**), после выбора открывает форму (Modal) на **5 вопросов**, а затем отправляет заявку в канал на рассмотрение красивым Embed.

## Установка

1) Поставь **Python 3.11+** (на Windows лучше через официальный установщик).

2) Установи зависимости:

```bash
py -m pip install -r requirements.txt
```

3) Создай файл `.env` рядом с `main.py` (можно скопировать из `.env.example`) и заполни:

- `DISCORD_TOKEN`
- `APPLICATION_CHANNEL_ID`
- (опционально) `PANEL_CHANNEL_ID`
- (опционально) `GUILD_ID`

## Запуск

```bash
py main.py
```

## Как пользоваться

- В Discord вызови команду `/panel` (нужны права администратора), чтобы отправить панель в канал.
- Пользователь выбирает тип заявки и отвечает на 5 вопросов в форме.
- Бот отправляет заявку в `APPLICATION_CHANNEL_ID` и (если может) создаёт ветку под заявкой.

## Настройка вопросов

Вопросы лежат в `ui.py` в списках `ACADEMY_QUESTIONS` и `YOUNG_QUESTIONS`.
